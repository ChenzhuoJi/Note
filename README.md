lines. md: lines本意是台词，我想，一本书讲的故事，就是一段旁白。于是不管是书中的语言，还是影视上的话，我都称之为lines。

novel. md: 古典乐和写作是秘方，我现在教给你，它就是你必须要知道的配方了。
